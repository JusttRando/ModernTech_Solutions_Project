## How to push a file 

1. git add.
2. git commit -m "(message)"
3. git push -u origin (branch name)

## How to enter a different branch 

first make sure your in a different branch. Look at the bottom left to see the branch name.

1. git checkout (branch name)

## How to get changes made by the group 

please be in the main branch when doing this 

1. git pull 

## Remove a file 

1. git rm (file name)
